## 📋 خطة الاختبار اليدوية

### الخطوة 1: تشغيل الخادم
```bash
npm run dev
```
يفتح على: `http://localhost:5173`

### الخطوة 2: اختبار تسجيل الدخول
1. انتقل إلى صفحة تسجيل الدخول
2. أدخل: `demo@demo.com` / `123456`
3. تحقق من الوصول إلى لوحة القيادة

### الخطوة 3: اختبار لوحة القيادة
1. تحقق من تحميل جميع الرسوم البيانية
2. تحقق من الأرقام المعروضة
3. تفاعل مع المخططات

### الخطوة 4: اختبار القيود المحاسبية
1. انتقل إلى `القيود المحاسبية`
2. انقر على "قيد جديد"
3. أضف سطراً بـ:
   - الحساب: اختر من القائمة
   - المدين: 100
4. أضف سطراً آخر:
   - الحساب: آخر
   - الدائن: 100
5. احفظ وتحقق من الرسالة

### الخطوة 5: اختبار الفواتير
1. انتقل إلى `الفواتير البيعية`
2. أنشئ فاتورة جديدة
3. أضف عميل وعنصر
4. اعتمدها

### الخطوة 6: اختبار المخزون
1. انتقل إلى `المخزون`
2. فرز وابحث
3. تحقق من الكميات

### الخطوة 7: اختبار إدارة المستخدمين
1. انتقل إلى `إدارة المستخدمين`
2. جرب إنشاء مستخدم جديد
3. جرب حذف (للديمو فقط)

### الخطوة 8: اختبار الإعدادات
1. انتقل إلى الإعدادات
2. غير اللغة
3. تحقق من التأثيرات
