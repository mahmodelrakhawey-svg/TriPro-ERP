# 📖 التوثيق الشامل لمشروع TriPro-ERP

## 1. نظرة عامة على المشروع (Project Overview)
نظام ERP متكامل مبني باستخدام React, TypeScript, و Supabase. يغطي النظام مديولات المحاسبة، المخازن، التصنيع، المقاولات، وإدارة المطاعم (POS/KDS).

### الحالة الراهنة (Current Status):
*   **المحاسبة:** مكتملة (قيود، ميزانية، تدفقات نقدية).
*   **المخازن:** نظام متطور مع حساب التكلفة وإعادة حساب المخزون.
*   **التصنيع:** يدعم أوامر التشغيل، التكاليف المعيارية، وفحص الجودة.
*   **الأمان:** تم استبدال تنبيهات `alert` بنظام `showToast` وتأمين البيانات المحلية.

## 3. تعليمات تقنية وتشغيلية (Technical Guide)

### البدء السريع:
```bash
npm run dev      # تشغيل بيئة التطوير
npm run build    # بناء نسخة الإنتاج مع فحص أمني
npm run test     # تشغيل اختبارات الجودة (75 اختبار حالياً)
```

### إعادة بناء قاعدة البيانات (Order of Scripts):

## 4. معايير الكود والأمان (Best Practices)

### معالجة الأخطاء:
يُمنع استخدام `alert()`. استخدم دائماً:
```typescript
const { showToast } = useToast();
showToast('Message', 'success' | 'error');
```

### الأمان:
*   لا تضع كلمات مرور في الكود (Hardcoded).
*   استخدم `process.env.NODE_ENV === 'development'` عند استخدام `console.log`.
*   البيانات المحلية تُخزن عبر `secureStorage`.
